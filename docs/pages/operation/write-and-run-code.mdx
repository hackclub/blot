> ⚠️ Note:
> You'll need to use a browser that supports the Web Serial API. The editor is known to be compatible with the latest version of Chrome (version 89 or above). Other browsers that may be comaptible are listed [here](https://developer.mozilla.org/en-US/docs/Web/API/Web_Serial_API#browser_compatibility).

# Write and run code in the browser

1. Open the [editor](https://editor.haxidraw.hackclub.com/)
2. Click the "connect" button to connect your machine to the editor. Be sure your machine is plugged in and that you have the latest firmware.
3. Write code in the editor on the left and click the "run" button to execute your code.
4. Save your code locally with the "download" button.
---
Alternatively, you can run the interface locally. go to ``..\haxidraw\interface`` and run:
```
yarn
yarn dev
```
in order to launch the interface.
